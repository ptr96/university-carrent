Projekt
Programowanie Obiektowe - JAVA
Projekt jest pamięciową bazą danych budowaną na liście, ma on kilka funkcjonalnosci.
możemy dodawać poszczególne obiekty(użytkowników, samochody, tiry), 
sprawdzać przypisania. edytować bazę, przeszukiwać ją. i tym podobne.
W bazie rejstrujemy samochody (marka, model, rodzaj paliwa) oraz tiry. Przypisujemy im użytkowników, 
bądź pozostawiamy w bazie jako pojazdy wolne - w wypożyczalni.
Klasy dziedziczą własnosci po abstrakcyjnej klasie głównej - DataBaseObject- jest to główne założenie projektu.
Projekt miał przedstawiać podstawowe działanie bazy systemu wypożyczalni samochodów/tirów.
Jest to aplikacja konsolowa.
Dodatkowo folder zawiera plik uml.drawio, jest to plik diagramu UML przedstawiający strukturę programu.
